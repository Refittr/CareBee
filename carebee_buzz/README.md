# BUZZ by CareBee - Mobile App

This is the mobile companion app for the CareBee ecosystem, built with Flutter.

For a comprehensive overview of the project architecture, features, and setup instructions, please refer to the **[Main Project README](../README.md)**.

## Key Features
- **Achiever Mode**: High-accessibility tablet interface.
- **Supporter Mode**: Real-time management of routines and nudges.
- **Supabase Integration**: Auth, Realtime, and Postgres.

## Mobile Specifics
- **Framework**: Flutter
- **State Management**: Riverpod
- **Routing**: GoRouter
